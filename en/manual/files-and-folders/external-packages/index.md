# External packages

<span class="badge text-bg-primary">Intermediate</span>

In Stride, it is possible to use external [Nuget](https://www.nuget.org/) packages in your project. They can contain not only code, but also their own **assets and resources**.

This system allows you to create and share your own packages for Stride with other people, as well as access those made by others.
