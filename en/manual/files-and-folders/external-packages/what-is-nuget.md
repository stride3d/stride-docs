# What is Nuget?

Nuget is a package manager for .NET projects (which includes C#). It's used by Stride to manage engine libraries and it can also be used for utilizing other ones in your game.

**Nuget isn't exclusive to Stride**. It's used in a lot of other C# and .NET projects that have nothing to do with the engine. You can easily take advantage of that to use any publicly available library in your game.

For more information on Nuget, check out [Nuget for Beginners](https://github.com/lukepadiachy/nuget-for-beginners/blob/main/README.md).
