# UWP

> [!WARNING]
> **UWP is no longer being developed**. We recommend building the game for [Windows](../windows/index.md) instead.

Universal Windows Platform (UWP) provides a common app platform available on every device that runs Windows. For more information, see [Intro to the Universal Windows Platform](https://docs.microsoft.com/en-us/windows/uwp/get-started/universal-application-platform-guide).

In October 2021 it was announced that UWP will stop receivng new features. Stride's support only exists for compatibility reasons.
