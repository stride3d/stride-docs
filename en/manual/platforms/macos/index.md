# MacOS

**MacOS** is a desktop operating system used by [Apple](https://apple.com) devices.

## Development support

Currently, MacOS is only **partially supported**. It's not possible to run **Game Studio**  (the editor) on it.

However, it is possible to code and build games natively, meaning that when in a team, some programmers could work on a project using MacOS for tasks that don't require the editor.

[!INCLUDE [xplat-editor-notice](../../../includes/xplat-editor-notice.md)]

Alternatively, it is possible to create Stride games on MacOS using a [code-only](https://stride3d.github.io/stride-community-toolkit/manual/code-only/index.html) approach, removing the need for an editor all together.

## General support

Stride's MacOS support is in an experimental state. You might encounter problems that may not be easily solvable without modifications to the engine.
