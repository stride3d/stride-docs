# iOS

**iOS** is an operating system used by mobile devices made by [Apple](https://apple.com).

## General support

Starting with 4.4, iOS is no longer in an experimental state. **All engine features are fully supported on it**, but Stride remains more focused on the desktop side and may be lacking some mobile-specific functionality.

However, **there aren't currently many people looking after iOS**. The engine is tested on the platform before any change is merged to the source code, but **sometimes some regressions could slip by**, requiring manual testing from active maintainers. If you want to make sure that all engine features continue being supported on it, consider [becoming a contributor](../../../contributors/index.md).

> [!NOTE]
> All iOS applications (including games made with Stride) **can only be built on devices running [MacOS](../macos/index.md)**.
